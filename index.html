Arrays

An array is a data structure that stores a collection of elements, typically of the same data type, in a contiguous block of memory. Arrays allow you to efficiently store and retrieve data using an index.

Types of Arrays:
  • One-dimensional Array (1D): A simple list of elements, where each element can be accessed using a single index. Example: int arr[] = {1, 2, 3, 4, 5};
  • Two-dimensional Array (2D): An array of arrays, often referred to as a matrix or table, where you can access elements using two indices (row and column). Example: int arr[2][3] = {{1, 2, 3}, {4, 5, 6}};
  • Multi-dimensional Array: Arrays with more than two dimensions, such as 3D, 4D, etc. Example: int arr[2][3][4];
  • Jagged Array: An array of arrays where each inner array may have different lengths. Example: int[][] arr = new int[3][];
  • Dynamic Array: An array whose size can change during runtime. Example: std::vector<int> arr;

Stacks and Queues

Stacks and Queues are fundamental data structures used in programming. A stack follows the LIFO (Last In, First Out) principle, whereas a queue follows the FIFO (First In, First Out) principle.

1. Stack

A stack is a linear data structure where elements are inserted and removed from the same end (the top). It follows the LIFO principle, meaning the last element added is the first to be removed.

Operations on Stack:
  • Push(x): Insert an element at the top.
  • Pop(): Remove and return the top element.
  • Peek()/Top(): Get the top element without removing it.
  • isEmpty(): Check if the stack is empty.
  • isFull(): Check if the stack is full (for fixed-size stacks).
Implementation of Stack in C++ (Using Array)

                                #include <iostream>
                                using namespace std;
                                
                                #define SIZE 5  // Stack size
                                
                                class Stack {
                                    int arr[SIZE];  
                                    int top;  
                                
                                public:
                                    Stack() { top = -1; }  
                                
                                    void push(int x) {
                                        if (top == SIZE - 1) {
                                            cout << "Stack Overflow\n";
                                            return;
                                        }
                                        arr[++top] = x;
                                    }
                                
                                    void pop() {
                                        if (top == -1) {
                                            cout << "Stack Underflow\n";
                                            return;
                                        }
                                        top--;
                                    }
                                
                                    int peek() {
                                        if (top == -1) {
                                            cout << "Stack is Empty\n";
                                            return -1;
                                        }
                                        return arr[top];
                                    }
                                
                                    bool isEmpty() {
                                        return (top == -1);
                                    }
                                };
Applications of Stack:
  • Function Calls (Call Stack)
  • Undo/Redo Operations
  • Expression Evaluation (Infix to Postfix Conversion)
  • Backtracking (Maze problems, Recursion)
2. Queue

A queue is a linear data structure where elements are inserted at the rear and removed from the front. It follows the FIFO principle.

Operations on Queue:
  • Enqueue(x): Insert an element at the rear.
  • Dequeue(): Remove and return the front element.
  • Front(): Get the front element without removing it.
  • Rear(): Get the last element without removing it.
  • isEmpty(): Check if the queue is empty.
  • isFull(): Check if the queue is full (for fixed-size queues).
Implementation of Queue in C++ (Using Array)

                                #include <iostream>
                                using namespace std;
                                
                                #define SIZE 5  
                                
                                class Queue {
                                    int arr[SIZE];  
                                    int front, rear;  
                                
                                public:
                                    Queue() { front = rear = -1; }
                                
                                    void enqueue(int x) {
                                        if (rear == SIZE - 1) {
                                            cout << "Queue Overflow\n";
                                            return;
                                        }
                                        if (front == -1) front = 0;
                                        arr[++rear] = x;
                                    }
                                
                                    void dequeue() {
                                        if (front == -1 || front > rear) {
                                            cout << "Queue Underflow\n";
                                            return;
                                        }
                                        front++;
                                    }
                                
                                    int getFront() {
                                        if (front == -1 || front > rear) {
                                            cout << "Queue is Empty\n";
                                            return -1;
                                        }
                                        return arr[front];
                                    }
                                
                                    bool isEmpty() {
                                        return (front == -1 || front > rear);
                                    }
                                };
Types of Queues:
  • Simple Queue: Follows FIFO, inserts at the rear, removes from the front.
  • Circular Queue: The last position is connected to the first position to utilize space efficiently.
  • Deque (Double-Ended Queue): Allows insertion and deletion from both front and rear.
  • Priority Queue: Elements are removed based on priority rather than order of insertion.
Applications of Queue:
  • CPU Scheduling (Round Robin Scheduling)
  • Breadth-First Search (BFS) Algorithm
  • Printer Queue / Call Center Requests
  • Cache Implementation
Key Differences Between Stack and Queue:
Feature Stack (LIFO) Queue (FIFO)
Order Last In, First Out First In, First Out
Operations Push, Pop, Peek Enqueue, Dequeue, Front
Insertion Insert at Top Insert at Rear
Deletion Remove from Top Remove from Front
Use Cases Recursion, Undo/Redo, Expression Evaluation BFS, Scheduling, Resource Allocation

Linked Lists

A linked list is a linear data structure where each element is a separate object called a node. Each node contains data and a reference (or link) to the next node in the sequence.

Types of Linked Lists:
  • Singly Linked List: Each node points to the next node. Example: struct Node { int data; Node* next; };
  • Doubly Linked List: Each node points to both the next and previous nodes. Example: struct Node { int data; Node* next; Node* prev; };
  • Circular Linked List: The last node points back to the first node, forming a circle. Example: struct Node { int data; Node* next; };

Example:

State and Props

Props are used to pass data from parent to child components, while State is used to manage data within a component.

Example:
class Clock extends React.Component {
                            constructor(props) {
                                super(props);
                                this.state = { date: new Date() };
                            }
                            render() {
                                return (
                                    <div>
                                        <h1>Current Time: {this.state.date.toLocaleTimeString()}</h1>
                                    </div>
                                );
                            }
                        }

Hooks

Hooks allow you to use state and other React features in functional components.

Example:
import React, { useState } from 'react';
                        
                        function Counter() {
                            const [count, setCount] = useState(0);
                            return (
                                <div>
                                    <p>You clicked {count} times</p>
                                    <button onClick={() => setCount(count + 1)}>Click me</button>
                                </div>
                            );
                        }
